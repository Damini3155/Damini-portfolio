## 🛠️ Installation

To run this project locally:

1. **Clone the repository**  
   ```bash
   git clone https://github.com/Damini3155/Damini-portfolio.git
````

2. **Navigate into the folder**

   ```bash
   cd Damini-portfolio
   ```
3. **Install dependencies**

   ```bash
   npm install
   ```
4. **Start the development server**

   ```bash
   npm start
   ```
